How to play?

Memory Game is game for one player in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.  The theme of this game is The Eight Summits - "collective name for eight highest mountains in the seven continents".
On loading the game the player has to type a nickname. After this the player will see a deck of 16 cards, 4x4, laid face down. Above the deck can be find the score panel, starting with the player’s name, stars rating of the performance on the base of the number of made moves, the number of the flips, timer in seconds and restart button. The game has 3 levels of difficulty During the first game, which is also first level of the game, the player has 60 seconds to pair the matching cards. On successful completion, the player continues to the second level in which the time decrease to 55 seconds, after which continues to the last third level - 50 seconds.
